## 業務効率化・自動化の主な案件実例

### 1. 求人系広告代理店:レポーティング業務の全自動化
従来 Excel VBA で作成していた定例レポートを、Python on Cloud Run と BigQuery を組み合わせて全自動化。スプレッドシートからのデータ連携も統合し、担当者の手作業を大幅に削減しました。

**使用技術:** Python / Cloud Run / BigQuery / Google Sheets

### 2. 大手通信会社:ピープルアナリティクス基盤の構築
分析官が各自個別に行っていた人事データの前処理を ETL 処理として一元化。bat 処理で自動実行可能にし、人事本部の業務効率化を実現しました。機微な人事データの運用要件もロジカルに整理し、運用ドキュメントを整備。

**使用技術:** AWS / Python / Excel

### 3. 自動車業界:IoT データの自動可視化
SQLServer から BigQuery への DB 移管、GCP 上のサーバー環境構築を担当。GPS 情報の IoT データを Looker Studio で自動可視化し、業務担当者がリアルタイムで状況把握できる体制を整備。同一クライアントから別案件も継続受注。

**使用技術:** GCP / BigQuery / Cloud Composer / Looker Studio / RShiny / Python

### 4. アパレル業界:DX 化支援
BigQuery + Airflow での ETL 開発・運用保守を担当。Looker Studio でのダッシュボード構築により、業務担当者のデータ活用を促進しました。運用保守を他社から引き受け、クライアントから指名で信頼を獲得。

**使用技術:** GCP / BigQuery / Airflow / Looker Studio

### 5. 飲料メーカー:8年間未分析だったデータの活用支援
リリースから8年間、ほぼ未分析だったユーザーデータの分析を実施。決定木分析・クラスタリングによる顧客セグメンテーションを行い、サービスの継続判断にデータで貢献。存続危機だったサービスを継続決定に導きました。

**使用技術:** Snowflake / Python / KNIME

### 6. グローバル製造業:オンプレ→クラウド移行
OracleDB / Dr.SUM から Snowflake / Tableau Online へのデータ分析基盤の移行を担当。PoC から長期プロジェクトへと発展。

**使用技術:** Snowflake / Tableau / OracleDB / Dr.SUM
